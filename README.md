# Cinel Python Training 🐍

This repository contains all the exercises and projects developed during the Python training at Cinel. It covers everything from basic logic to file manipulation and automation.

## 📂 Project Structure

*   **pythonProject / pythonProject2:** Core logic exercises and problem solving.
*   **teste607:** Advanced exercises involving classes and file handling (.txt, .csv).
*   **File_Organizer:** (Example) Automation script to manage downloads/folders.

## 🛠️ Skills Applied
- **File I/O:** Reading and writing data in multiple formats.
- **Logic:** Complex algorithm development.
- **Best Practices:** Clean code and organized repository structure.
